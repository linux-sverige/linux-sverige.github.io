# EUnion - European Union Cloud

## Commercial Website

This is the official commercial website for EUnion (european-union.cloud) - Europe's GDPR-native alternative to Office 365.

### Brand Identity

**Slogan:** "Your Data, Your Union"

**Core Values:**
- GDPR-Native by Design
- EU Data Sovereignty
- No CLOUD Act
- European Independence

**Colors:**
- Primary Blue: `#003399` (EU Blue)
- Accent Gold: `#FFCC00` (EU Gold)
- Text Dark: `#1a1a2e`
- Background Light: `#f5f5f5`

### Pages

1. **index.html** - Landing page with hero section and value proposition
2. **pricing.html** - Pricing plans and comparison
3. **features.html** - Detailed features overview
4. **about.html** - About EUnion, trust, and GDPR commitment
5. **contact.html** - Contact form and information

### Technical Stack

- Pure HTML5, CSS3, JavaScript (vanilla)
- Responsive design (mobile-first)
- No external dependencies
- EU-hosted fonts (no Google Fonts)
- Self-contained assets

### Deployment

Website is designed to be hosted on:
- GitHub Pages
- Any static hosting
- EU-based servers preferred
