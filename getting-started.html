🚀 Step 1: First Login

How to Log In

  1. Open your web browser
  2. Go to: https://disk.your-cloud.org/login
  3. Enter the username and password you created during registration
  4. Click Log in
💡 Tip: Bookmark this page for easy access!

Can't Log In?

  • Forgot password? Click "Forgot password?" on the login page
  • Didn't receive confirmation email? Check your spam/junk folder
  • Still having issues? Contact us at: info@linux-sverige.se

👤 Step 2: Set Up Your Profile

Update Your Profile Picture

  1. Click on your profile picture (top-right corner)
  2. Select Settings
  3. Click Profile picture
  4. Upload a photo from your computer

Set Your Display Name

  1. In Settings, click Personal info
  2. Enter your full name in the Display name field
  3. This name will be shown when you share files or send calendar invites

Add Your Existing Email

Note: EUnion currently focuses on file storage and calendar. For email, you can:
1. Continue using your existing email provider (Gmail, Outlook, etc.)
2. Add your email address to your profile for notifications:
   - Go to Settings → Personal info
   - Add your email address
   - This ensures you receive important notifications about your account

📁 Step 3: Using Files (Cloud Storage)

Upload Your First File

  1. Click Files in the top menu
  2. Click the + button (or drag and drop files)
  3. Select Upload file or Upload folder
  4. Choose files from your computer

Create Folders

  1. Click the + button
  2. Select New folder
  3. Name your folder (e.g., "Work Documents", "Personal", "Projects")

Share Files or Folders

Option 1: Share via Link (anyone can access)
1. Click the … (three dots) next to a file
2. Select Share
3. Click + Share link
4. Copy the link and send it to anyone
Option 2: Share with specific people
1. Click the … next to a file
2. Select Share
3. Type the email address of another EUnion user
4. Set permissions (Can view / Can edit)
5. Click Share
🔒 Security Note: All files are stored securely in the EU and protected by GDPR.

📅 Step 4: Using Calendar

Access Your Calendar

  1. Click Calendar in the top menu
  2. You'll see your personal calendar

Create an Event

  1. Click on a date/time
  2. Enter event title
  3. Set start and end time
  4. Add location (optional)
  5. Add description (optional)
  6. Click Save

Import Your Existing Calendar

From Google Calendar:

  1. In Google Calendar, go to Settings → Import & export
  2. Export your calendar as .ics file
  3. In EUnion Calendar, click Settings (bottom-left)
  4. Click Import calendar
  5. Select your .ics file

From Outlook:

  1. In Outlook, go to File → Open & export → Import/Export
  2. Export to a file (.ics format)
  3. Import to EUnion as above

Connect Calendar to Your Existing Email

To receive calendar invites on your current email:

  1. Share your EUnion calendar with your existing email address
  2. Or export your EUnion calendar and import to your current email provider

📧 Step 5: Using EUnion with Your Existing Email

Current Setup

EUnion provides:
✅ Cloud file storage
✅ Calendar with sync
✅ Secure EU data storage

For email: Continue using your existing email provider (Gmail, Outlook, etc.)

How to Stay Connected

Option 1: Use EUnion Files with email attachments
- Upload files to EUnion
- Share via link in your regular emails
- Recipients get secure access without registration
Option 2: Calendar invites
- Create events in EUnion Calendar
- Add attendees using your existing email address
- Invites sent from your current email provider
Option 3: Email notifications
- Add your email to EUnion profile (Settings → Personal info)
- Get notifications about shared files and calendar events

📱 Step 6: Install Mobile Apps

Nextcloud App (for Files)

iPhone/iPad:

  1. Go to App Store
  2. Search "Nextcloud"
  3. Download and install
  4. Open app → Log in
  5. Enter server address: https://disk.your-cloud.org
  6. Enter your username and password

Android:

  1. Go to Google Play Store
  2. Search "Nextcloud"
  3. Download and install
  4. Same login steps as above

Calendar Sync

The Nextcloud app will sync your calendar automatically once logged in.

Alternative: DAVx⁵ (for advanced users)

For separate calendar/contacts sync:

  1. Install DAVx⁵ from app store
  2. Add account with URL: https://disk.your-cloud.org
  3. Sync calendars and contacts

💡 Pro Tips

Keyboard Shortcuts

  • C - Create new file/folder
  • U - Upload file
  • ? - Show all shortcuts

Storage Management

  • Check your storage usage: Settings → Personal info
  • Empty trash regularly to free space
  • Use External storage to connect other cloud services

Security Best Practices

  1. ✅ Use a strong, unique password
  2. ✅ Enable two-factor authentication (in Settings → Security)
  3. ✅ Log out when using shared computers
  4. ✅ Don't share sensitive files via public links

Common Questions

Q: Can I access my files offline?
A: Yes! Install the Nextcloud Desktop app to sync files to your computer.

Q: How much storage do I have?
A: Check Settings → Personal info → Storage usage

Q: Can I share with non-EUnion users?
A: Yes! Use Share link feature and send them the link.

Q: Is my data really in the EU?
A: Yes! All data is stored on servers in Europe and protected by GDPR.

Q: Why can't I send email from EUnion?
A: EUnion currently focuses on file storage and calendar. Use your existing email provider and share EUnion files via links.

⏰ What Happens After 30 Days?

Day 25: Reminder

You'll receive an email reminder that your trial ends in 5 days.

Day 30: Trial Ends

Option 1: Upgrade to Paid
• Keep all your files, emails, and calendars
• Continue using EUnion with a subscription
• Contact us: info@linux-sverige.se
Option 2: Trial Expires
• Your account will be deactivated
• Download your data before the trial ends
💡 Tip: We recommend downloading important files before day 30 if you choose not to continue.
